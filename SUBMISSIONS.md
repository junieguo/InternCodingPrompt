# Submission Instructions

Please follow the steps below to submit your completed coding exercise.

## 1. Fork the Repository

Create a fork of this repository in your own GitHub account.

## 2. Complete the Exercise

Make your changes in your forked repository and ensure your solution includes all required deliverables described in `README.md`.

Before submitting, make sure you:

- Run the application locally
- Run the unit tests successfully
- Commit all relevant changes to your fork
- Push your final branch to GitHub

## 3. Prepare Your Submission

Verify that your fork contains:

- Your completed implementation
- Any updated project files required by the exercise
- A clear commit history with your final solution pushed to GitHub

## 4. Email Your Submission

Send an email with the link to your finalized forked repository.

Include the following in your email:

- Your full name
- A link to your forked repository
- The branch name containing your final submission
- Any brief notes you would like us to consider while reviewing

## 5. Final Check

Before sending your email, confirm that:

- The repository link is accessible
- Your latest changes have been pushed
- The submitted branch contains your final version

Thank you for completing the exercise.
